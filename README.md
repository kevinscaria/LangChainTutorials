# LangChain Tutorials
Basic to Advanced LangChain

This repository includes the following:
- Model Types
- PromptTemplates
- Creating Chains
- ChatAP for OpenAI models

## Get Started

1. Get your [OpenAI](https://platform.openai.com/signup) and [HuggingFace](https://huggingface.co/settings/tokens) API tokens.

> [!IMPORTANT]  
> Start by creating a `.env` file in the root directory of this repository clone.

2. In the .env file create the following KEYS:
```
HUGGINGFACE_API_TOKEN=<YOUR_API_TOKEN>
OPENAI_API_KEY=<YOUR_OPENAI_API_KEY>
```
3. Run the notebooks
